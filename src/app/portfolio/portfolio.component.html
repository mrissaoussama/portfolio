I made a mobile app and a server for a parking solution that was local.

                    
                    
this project was made using the .Net framework. The server is made in ASP.Net and is deployed on Azure. It receives info from the
                      local solutions using a plugin made for the in-house software and notifies the relevant users about earnings, statistics, and warnings on their Android or Apple devices.
                    Notifications are sent and received in real time using SignalR. the connections are secure with JWTs
